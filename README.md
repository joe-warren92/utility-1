# Utility 1

Frontend-only Astro landing site for Utility 1 Ltd.

## Scripts

- `npm install` installs dependencies.
- `npm run dev` starts the local development server.
- `npm run build` builds the static site to `dist/`.
- `npm run preview` previews the production build locally.

## Project Structure

- `src/pages/` contains site pages.
- `src/components/` contains Astro section components.
- `src/styles/global.css` contains Tailwind v4 theme variables and global styles.
- `public/` contains static assets copied into the build.

## Forms

The contact and audit forms are static placeholders. Connect them to a hosted form provider or other external handling before launch.
